Dynamic Path Finder (AI Project)

Run on Fedora:
1) sudo dnf install -y python3 python3-tkinter
2) python3 main.py

What you can do:
- Build grid (rows/cols)
- Place Start and Goal
- Draw walls or generate random walls
- Run GBFS or A* with Manhattan/Euclidean
- See Frontier (yellow), Visited (blue), Final path (green)
- Dynamic Mode: obstacles can spawn while agent moves, and agent replans
